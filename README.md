# 🧹 SmartClean AI – Data Cleaning Tool

SmartClean AI is an interactive web application built using **Streamlit** that helps users clean, analyze, and understand datasets efficiently.

🔗 **Live App:** https://data-cleaning-tool.streamlit.app  
💻 **GitHub Repo:** https://github.com/Rishabh25-Bot/data-cleaning-tool

---

## 🚀 Features

- 📂 Upload datasets (CSV, TXT, Excel)
- 📊 Dataset overview (rows, columns, missing values, duplicates)
- 📌 Data profiling with summary statistics
- 📉 Missing values visualization
- 📈 Correlation heatmap
- 🚨 Outlier detection (IQR method)
- 🔍 Column-wise analysis
- 🧼 Data cleaning:
  - Remove duplicates
  - Handle missing values
  - Remove high-missing columns
- 📥 Download cleaned dataset

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit
- Pandas
- Plotly
- Seaborn
- Matplotlib
